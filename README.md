# RPG-Generator
RPG Generator Web App made using Flask

## Setup:
1. .env configuration:
```python
DB_CONNECTION_STRING=sqlite:///path/to/database.db
```

2. Python libs:
```bash
pip install -r requirements.txt
```

3. DB Setup:
```bash
python db_setup.py
```

4. Run with:
```bash
flask run --debug
```
